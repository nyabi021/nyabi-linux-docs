# nyabi-system-docs
A collection of personal troubleshooting logs and configuration notes for Linux and macOS environments.

## Focus
- **OS**: EndeavourOS, macOS (Apple Silicon)
- **WM/DE**: GNOME, Hyprland

## Structure
- `/troubleshooting`: Documentation of symptoms, analysis, and resolution procedures.
- `/configurations`: Management of dotfiles and service configuration files.
- `/scripts`: Scripts for system automation and setup.

## Documentation Policy
- Includes both resolved and unresolved technical issues.
- Records practical solutions and results derived from external sources.
- Minimizes narrative descriptions to focus on technical data and commands.
